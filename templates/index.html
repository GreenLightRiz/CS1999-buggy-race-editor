WELCOME TO THE BUGGY EDITOR

Use this editor to specify a racing buggy. The editor saves it in its little database and generates JSON data for the buggy. This is the data you need to supply when you log into the race server and enter your buggy into the next race.

HOME takes you to this page, which is the main menu!

CREATE BUGGY takes you to an interface where you can customise your own buggy!

YOUR BUGGY SPECIFICATIONS takes you to a page where the data you made for your buggy is displayed in a tabulated format!

GET BUGGY JSON takes you to a page where the data you made for your buggy is shown as a JSON file!

INFO gives you all the information you need to know about the buggy you're making

POSTER takes you to a page of a poster I made which describes my process in coding for this project

Remember that if your data is not accepted by the race server, your buggy will be disqualified from that race - so make sure you program your editor correctly.
